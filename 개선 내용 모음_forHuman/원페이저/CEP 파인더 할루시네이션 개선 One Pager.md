# CEP 파인더 할루시네이션 개선 One Pager

## Why

지금의 CEP 파인더는 [기존 기획안](https://docs.google.com/presentation/d/1z3uzJDYg6iqxSB_AfdCvHalguCEW5QB6sBcek4oO2sU/edit?slide=id.g3e5653b7e83_0_107#slide=id.g3e5653b7e83_0_107)에서 그린 방향을 제대로 따라가지 못하고 있다.

Obj3. Data Differentiation : 데이터 제공방식 혁신 및 신뢰성 제고  

### Background 

현재 기존 21개의 프로젝트 중 각 카드는 다음과 같은 [할루시네이션 지표](https://hallucination-test.vercel.app/dashboard)를 보였다.

- 실패 11건 (5.2%)
- 의심 83건 (39.5%)

## What

위 실패/의심 사례를 해결하기 위해 다음 작업을 진행하고자 한다.

1. CEP 파인더 Phase 내 프롬프트 개선 작업 (非 강제성)
2. 대시보드 내 전체 정합 수치 70% 이상으로 향상 목표

## How

### CEP 파인더 Phase 내 프롬프트 개선 작업 (非 강제성)

Phase 3와 4를 중심으로, 다음 사항을 개선한다. 필요 시 후속으로 이어지는 Phase도 개선을 한다.

|  | **위치** | **이유** |
| --- | --- | --- |
| 1 | Phase 3 | 상황이 인용 URL에서 발견되지 않아도 강제로 생성 |
| 2 | Phase 3 | 의역이 발생함 |
| 3 | Phase 4 | 앞서 만들어진 7W 요소가 강제적으로 사용되게 됨. |

스킬 기반 `cep-prompt-improver` 할루시네이션 검증 스킬을 만들어 처리.

- CEP파인더 json 전달
- 할루시네이션 검수
- 검수 결과를 근거로 Phase 프롬프트 자가 개선

### 대시보드 내 전체 정합 수치 70% 이상으로 향상 목표

[CEP 대시보드](https://hallucination-test.vercel.app/dashboard)에서 전체 항목 정합률 70% 이상으로 목표를 설정하여, 이를 검증하여 CEP 파인더 서비스로 제공한다.
